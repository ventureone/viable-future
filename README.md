# The Viable Future

An interactive graph game about preserving the space of possible futures under irreversible time.

## What it is

A system can become more efficient, more coordinated, more predictable — and less able to change. What appears as strength can be a contraction. This game makes that contraction visible and playable.

You manage a graph of possible futures. Each turn you choose one of four operators:

- **Weave** — connect distant regions, thickening the interior
- **Fear** — harden the most exposed region, safer but less generative
- **Compression** — prune the weakest, clean and efficient
- **Regret** — reopen a recent loss before it sediments

Meanwhile, sedimentation runs in the background. Unused paths grow brittle. The interior shrinks quietly. The game ends not in a crash but when H(t) reaches zero — when every surviving node faces the void and no depth remains. The graph still exists. Nothing can change.

## The theoretical basis

Based on two notes by Rajendra Wadje (2025):

- *A Generative Grammar of Viable Futures*
- *Operator Semantics of Viable Futures*

These formalize viability as the preservation of reopenable future interior — a conserved invariant acted upon by a minimal operator algebra. The game is a playable interpretation of that framework.

## How to run

Download `index.html` and open it in any modern browser. No installation, no dependencies. It's a single self-contained file.
